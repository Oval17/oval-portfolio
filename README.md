# ovalele.in

Personal portfolio of **Anurag Singh** ([@oval](https://x.com/Oval170)) — live at [ovalele.in](https://ovalele.in)

## Stack

- [Next.js 15](https://nextjs.org/) — App Router, fully static
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS 4](https://tailwindcss.com/)
- [Motion](https://motion.dev/) — animations
- [Vercel](https://vercel.com/) — hosting, analytics, speed insights

## Pages

| Route | Description |
|---|---|
| `/` | Home — intro, experience, tech stack, GitHub activity |
| `/work` | My Lab — projects and builds |
| `/work/[slug]` | Individual project write-up |

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deployment

Deployed on [Vercel](https://vercel.com). Every push to `main` auto-deploys to [ovalele.in](https://ovalele.in).
